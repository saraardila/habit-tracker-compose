# 🌿 Habit Tracker

Minimal aesthetic habit tracker built with Kotlin & Jetpack Compose.

## ✨ Features
- Clean UI
- Smooth animations
- Checklist habits
- Material 3 design

## 🛠 Tech Stack
- Kotlin
- Jetpack Compose
- Room
- MVVM
- Hilt
